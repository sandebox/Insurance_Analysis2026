# 🛡️ AI Sales Intelligence — Insurance Analytics Platform

A predictive analytics SaaS platform for vehicle insurance, built with Streamlit.

---

## 📦 File Structure

```
ai_sales_intelligence/
├── app.py                  # Main Streamlit application
├── train.csv               # Dataset with 381k customers (included)
├── requirements.txt        # Python dependencies
├── README.md               # This file
└── utils/
    ├── __init__.py
    ├── data_utils.py       # Data loading, cleaning and filters
    ├── ml_utils.py         # ML models, scoring and rule-based insights
    ├── chart_utils.py      # All Plotly charts
    └── ai_utils.py         # OpenAI / LLM integration
```

---

## 🚀 How to Run Locally

### 1. Clone / unzip the project

```bash
cd ai_sales_intelligence
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv
source venv/bin/activate        # Linux/Mac
venv\Scripts\activate           # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the app

```bash
streamlit run app.py
```

Access at: **http://localhost:8501**

## ✨ Features

### 📊 Dashboard
- KPIs: total customers, conversion rate (12.26%), total pipeline (R$1.47B), average ticket
- Automatic opportunity alerts (hot leads)
- Age and Premium histograms by response
- Conversion by: Vehicle Damage, Vehicle Age, Age Group, Gender
- Full conversion funnel
- Variable correlation heatmap
- Average premium by segment

### 🤖 Machine Learning
- **Logistic Regression**: AUC = 0.809
- **Random Forest**: AUC = 0.844
- Metrics: Accuracy, AUC-ROC, Precision, Recall
- Confusion Matrix for each model
- Feature Importance with visual ranking
- Propensity score (0–100) for all 381k customers
- Tiers: 🔴 Low / 🟠 Regular / 🟡 Medium / 🟢 High / ⭐ Elite
- Full ranking CSV export

### 💡 Insights & Segments
- 6 automatic insights classified by impact (High/Medium)
- 4 priority segments with size, % of base and average ticket
- Channel recommendations per segment
- Cross-table: Vehicle × Damage × Conversion Rate

### ✨ AI Insights (OpenAI)
- Executive report generated by GPT-4o-mini
- Executive Summary, Opportunities, Risks, Recommendations
- 30-60-90 day Action Plan
- TXT report export
